Employee Management App

Technologies Used:

  • React - Used for front-end.
  • Spring- Used for back-end communication with database.
  • SQL - Database used to store employee information


Description:

This employee management application allows users to view their employees imformation such as their first name, last name and Email. It also allows users to add new employees, update their information and delete an employee from the database

Spring is used to establish connection with the database and make requests to the the database with the help of mySQL. React components are used for the front-end display and react-router is used to route to the various endpoints of the application
