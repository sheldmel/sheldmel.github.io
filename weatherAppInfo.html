Weather App

Technologies Used:

  • React - Used for front-end.
  • Node- Used for Back-end.
  • OpenWeatherMap API - Used to fetch weather and additional information

Description:

This weather application allows users to view information relating to the weather about a particular city such as temperature, forecast, wind speed, humidity, pressure etc. The user can type in the name of the city that they wish to view weather information of and the app displays the information realting to that city.

Axios is used to fetch the data based on the city entered by the user from a free to use API openweathermap.org which provides a range of weather-related products in a variable combination of depth and steps of measurement to millions of clients globally. React is used for handling the front-end and displaying the weather data while node is used to handle server reqests
