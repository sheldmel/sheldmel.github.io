Sudoku Game

Technologies Used:

  • Python - Used to create the solver.
  • PyGame - Used to display a sudoku grid

Description:

This app displays a soduku grid to users and allows to fill the grid and displays if the user solved the soduku puzzle or not. The algorithm used to solve the soduku puzzle is backtracking which is the principle of reverting back to the previous step or solution as soon as it is determined that the current solution cannot be continued. PyGame is used to help build the GUI of the solver and the amount to time spent solving the puzzle is also displayed to the user.
